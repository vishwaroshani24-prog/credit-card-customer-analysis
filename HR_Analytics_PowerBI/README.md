## HR Analytics Dashboard – Employee Attrition Analysis

### Project Overview
This Power BI dashboard analyzes employee attrition to identify trends across age, salary, job role and tenure.

### Tools Used
- Power BI
- DAX
- Excel

### Key Metrics
- Total Employees: 1.41K
- Attrition Count: 229
- Attrition Rate: 16%
- Average Age: 37
- Average Salary: 6.52K
- Avg Years at Company: 7.04

### Key Insights
- Highest attrition in 26–35 age group
- Employees earning up to 5k show highest attrition
- Sales Executive and Laboratory Technician roles have high attrition
